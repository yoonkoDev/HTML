기본태그 연습문제

각 문제는 Starter를 참고해 자유롭게 풀어보세요. 태그의 의미와 접근성(alt, label-for)을 지키는 것이 핵심!

문제 1 — 텍스트 기본

  1. h1 제목에 본인의 이름을 넣으세요.
  2. p 문단을 2개 이상 작성하고, strong과 em을 각각 1회 이상 사용하세요.
  3. 가운데 hr로 구분선을 넣으세요.
  4. 외부 링크를 하나 만들고, target="_blank"와 rel="noopener"를 적용하세요.

문제 2 — 이미지와 목록

  1. img로 좋아하는 사진을 넣고 alt를 적절히 작성하세요.
  2. ul 혹은 ol로 내가 좋아하는 음식/취미 Top 3를 작성하세요.
  3. mark, small, sup, sub를 한 번씩 사용해 보세요.

문제 3 — 표 만들기

  1. table로 간단한 가격표(또는 시간표)를 만드세요.
  2. 첫 번째 행은 th로 헤더를 구성하세요.
  3. 최소 3행 이상의 tr를 만들고, td로 데이터를 채우세요.

문제 4 — 폼 기초

  1. form 안에 이름/이메일 input과 메시지 textarea를 만드세요.
  2. label의 for 속성과 input/textarea의 id를 올바르게 연결하세요.
  3. button을 추가하고 버튼 텍스트를 “보내기”로 하세요.

도전 미션 — 종합 소개 페이지

  • 제목, 문단, 이미지, 링크, 목록, 표, 폼 요소 각각 1개 이상 포함
  • 접근성: alt, label-for 적용
  • 가독성: 섹션 구분과 hr 적절히 사용
  • (보너스) 창의적으로 small, mark, sup/sub 활용
